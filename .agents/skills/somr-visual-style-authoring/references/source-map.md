# Visual Style Source Map

Read these in this order unless the task is extremely narrow:

1. `docs/action-rpg-terminology.md`
   Use for the core gameplay-read vocabulary that the visuals must support.
2. `docs/art-direction-brief.md`
   Use for the master style lane, reference stack, and art approval checklist.
3. `docs/world-texture.md`
   Use for smaller peoples, professions, relic texture, and recurring regional detail.
4. `docs/visual-production-decisions.md`
   Use for benchmark priorities and what a benchmark scene must prove.
5. `docs/art/chapter-1-environment-production-kits.md`
   Use for Chapter 1 palette, material, prop, and dressing outputs.
6. `docs/planning/chapter-1-sightline-pass.md`
   Use when the task involves placeholder stills or story-stage visibility.
7. The relevant chapter doc and temple doc.
   Use them to keep scene art tied to story function and sacred context.

Prefer loading only the chapter and region you are actively designing for.
