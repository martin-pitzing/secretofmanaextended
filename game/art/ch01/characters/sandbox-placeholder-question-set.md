# Chapter 1 Sandbox Placeholder Question Set

Use these questions before drawing or generating each placeholder sprite.

## Core Questions

1. What is the single most important read?
   Examples:
   - `hero in practical travel wear`
   - `sacred institutional worker`
   - `displaced civilian`
   - `cute hostile creature`

2. What should the silhouette communicate before any detail is seen?

3. Which Chapter 1 region owns this character read?
   - `Potos`
   - `Forbidden Falls`
   - `Water Palace`
   - `Pandora route`

4. What is the one accent material or prop that sells the role?
   Examples:
   - rope charm
   - archive sash
   - travel bundle
   - ear shape

5. What palette family should dominate?

6. What palette family must stay secondary?

7. What detail should be withheld for now so the placeholder stays readable?

8. At gameplay distance, could this be mistaken for a different role?
   If yes, fix silhouette before anything else.
