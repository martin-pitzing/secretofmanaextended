# Art Folder

This folder will hold imported production assets and benchmark assets once the visual benchmark rooms and character sheets begin moving into implementation.

Until then, the art source of truth lives in:

- `docs/art-direction-brief.md`
- `docs/chapters/chapter-01-falls-to-pandora.md`

Placeholder review stills for Chapter 1 staging should live under:

- `game/art/ch01/review_stills/`

Chapter 1 placeholder character sprites for the reusable character sandbox should live under:

- `game/art/ch01/characters/`

The Chapter 1 story stages auto-detect `backdrop.*`, then `backdrop_v02.*`, then `backdrop_v01.*` inside each scene folder.
