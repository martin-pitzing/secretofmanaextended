# Rabite — canon reference sources

Drop one or more reference PNGs into this directory. The Rabite manifest
(`data/pipelines/monsters/forbidden_falls/rabite.yaml`) points at them via
`canon_references`.

## Recommended sources

- https://wikiofmana.com/wiki/Rabite — overworld sprite, multiple SoM games
- https://wikiofmana.com/wiki/Category:Rabite — variants worth inspecting

## What to keep when sourcing

- Clean transparent-background PNGs preferred
- Front-facing pose for the `idle_down` lane
- Side-facing pose for `idle_side`/`walk_side`/`attack_side` lanes if available
- Resolution: 64×64 minimum, 128×128+ preferred — Ludo upscales internally but
  starts from the reference's silhouette
