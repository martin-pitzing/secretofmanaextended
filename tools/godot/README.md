# Local Godot Binaries

Place local Godot executables in this folder so the workspace can launch the editor or run headless checks through:

- `tools/run-godot.ps1`

Expected examples:

- `Godot_v4.6.2-stable_win64.exe`
- `Godot_v4.6.2-stable_win64_console.exe`

Everything else in this folder is ignored on purpose, including local editor data created by the wrapper.
